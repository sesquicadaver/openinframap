Locust is a local OSINT infrastructure mapping tool built on OpenStreetMap data.

It visualises critical infrastructure including power grids, railways, pipelines, telecoms, water systems, ports, airports, and bridges.

Data

All geographic data is from OpenStreetMap contributors, licensed under the Open Database License (ODbL). Local exports are available on the exports page.

Stack

  • PostgreSQL / PostGIS — data storage
  • Imposm3 — OSM data import
  • Tegola — vector tile serving
  • MapLibre GL JS — map frontend
